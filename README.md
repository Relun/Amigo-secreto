# Amigo Secreto

Este es un proyecto en JavaScript para organizar un juego de Amigo Secreto. 

El juego permite:
  - Ingresar participantes.
  - Validar entradas.
  - Realizar un sorteo aleatorio.
  - Mostrar los resultados de manera segura.
